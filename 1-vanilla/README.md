# Shopping mall Searching page

## 작업환경

-   NodeJS
-   npm
-   lite-server
-   git

```
npm install lite-server -g
```

## 검색폼

[x] 검색 상품명 입력 폼이 위치한다. 검색어가 없는 경우이므로 x 버튼을 숨긴다
[x]검색어를 입력하면 x 버튼이 나온다
[x] 엔터를 입력하면 검색 결과가 보인다.
[x] x 버튼을 클릭하거나, 검색어를 삭제하면 검색 결과를 삭제한다.
